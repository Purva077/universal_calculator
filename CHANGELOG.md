# Changelog

All notable changes to Universal Calculator Hub are documented here.

## [3.0.0] — 2026-03-30

### Added
- PWA support — installable on Android, iOS, Windows, macOS
- Full offline support via Service Worker
- 8 new categories: House & Area, CS & Binary, Date & Time, All Converters, X→Y Equations, Sound & Frequency, Alphabet & Text, Science
- 60+ new calculators bringing total to 120+
- IEEE 754 float visualizer
- 1's / 2's / 9's / 10's complement calculators
- Ideal Gas Law, Half-Life Decay, Dilution, Molarity, pH
- Doppler Effect, Music Note Frequency, Decibel calculator
- Caesar Cipher, Word Value, Letter Position
- Room Area, Paint Calculator, Flooring Cost, Plot/Land Area
- Deep-link support via `?view=` query param
- Offline badge indicator
- PWA install prompt bar
- Keyboard support on standard calculator

### Improved
- Sidebar reorganized with section grouping
- Search now covers all 120+ calculators
- Analytics view with sample bar charts
- Mobile sidebar overlay with smooth animation

## [2.0.0] — 2025

### Added
- 60 calculators across 10 categories
- Dark/Light theme toggle
- Favorites (localStorage)
- History tracking
- Scientific calculator widget

## [1.0.0] — 2024

### Added
- Initial release with basic calculators
